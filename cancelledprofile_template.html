What Got Cancelled — Profile

The capacity that leaves the 2026–2028 window — cancelled outright, or delayed to 2029+ — profiled by deployment type, size, and proximity to the natural-gas pipeline network.

Scope: sites originally due 2026–28 that no longer land in the window — cancelled (far-future / indefinite model live date) plus delayed past 2028 (revised live date 2029+). Deployment type (neocloud / leasing / self-built) is the same labeled heuristic used on the Geography page (neocloud operator/tenant/GPU-cloud tag → neocloud; hyperscaler-type → self-built; else leasing). Pipeline proximity is straight-line distance to the nearest EIA gas pipeline — a proxy for how readily a site could add on-site gas generation. This is the model-derived set; the externally-cited cancellations (Indiana etc.) live on the Cancelled & Delayed page.
